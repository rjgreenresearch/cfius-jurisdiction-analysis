## Overview

This repository contains the replication code and data for "The Regulatory Perimeter: Counterfactual CFIUS Jurisdiction, Distance Discontinuities, and the Patchwork Problem in Foreign Agricultural Land Oversight" — the third article in a PhD by published work programme examining foreign agricultural land ownership near U.S. military installations.

**The paper asks a simple question:** Does the CFIUS real estate review regime — the federal government's primary mechanism for screening foreign agricultural land acquisitions near military installations — actually capture the acquisitions it is designed to screen?

**The answer:** Approximately 3.6 percent of the time.

### Headline Findings

| Finding | Result |
|---------|--------|
| Counties outside CFIUS jurisdiction (Regime 4) | 60 / 119 (50.4%) |
| Acreage outside CFIUS jurisdiction | 91,206 / 275,232 (33.1%) |
| Counties covered by 1-mile threshold (Part 1) | 0 / 119 (0.0%) |
| Part 1→Part 2 reclassification coverage gain | +25.2 percentage points |
| Compound detection probability (with Article 2) | 3.6% |
| Marginal counties per new site (Regime 4) | 0.10 |
| Coverage regression from Iowa site replacement | −35,867 acres |

## Reproducing the Results

### Prerequisites

```bash
pip install openpyxl pyshp numpy scipy
```

### Required Input Data

Place the following files in `data/inputs/`:

| File | Source | URL |
|------|--------|-----|
| `AFIDACurrentHoldingsYR2024.xlsx` | USDA Farm Service Agency | [fsa.usda.gov](https://www.fsa.usda.gov/programs-and-services/economic-and-policy-analysis/afida/index) |
| `afida_current_holdings_yr2018.xlsx` through `yr2023.xlsx` | USDA FSA (same page) Annual files for DiD panel | [fsa.usda.gov](https://www.fsa.usda.gov/resources/economic-policy-analysis/afida/annual-reports-underlying-data)
| `c_16ap26.zip` | NOAA Geographic Reference - County centroids shapefile | https://www.weather.gov/gis/Counties
| `mirta-dod-sites-boundaries-geojson.geojson` | DataLumos Project 239602 | [datalumos.org](https://www.datalumos.org/datalumos/project/239602/version/V1/view) |

### Execution Order

```bash
cd scripts/

# Task 1a: Build regime-tagged site database (230 entries)
python build_cfius_db.py

# Tasks 1b–2: Geocode sites + classify jurisdiction (119 counties × 4 regimes)
python article3_cfius_analysis.py

# Task 4: Multi-threshold boundary discontinuity (25, 50, 75, 100, 125 mi)
python article3_rdd.py

# Task 5: State-level difference-in-differences (51 states × 7 years)
python article3_did.py

# Task 6: Complexity-proximity analysis (63 matched + 56 opaque)
python article3_complexity.py
```

Total runtime: approximately 60 seconds on commodity hardware.

## Data Descriptions

### CFIUS Appendix A Site Database (`cfius_appendix_a_geocoded.csv`)

230 military installations from 31 CFR Part 802 Appendix A, digitized from three Federal Register publications (85 FR 3166, 88 FR 57350, 89 FR 88133) and verified against the current eCFR text.

| Field | Description |
|-------|-------------|
| `site_name` | Installation name (current eCFR designation) |
| `location` | City/state from Appendix A |
| `current_part` | 1 (1-mile threshold), 2 (100-mile), or 0 (removed) |
| `threshold_miles` | 1 or 100 |
| `regime_added` | 2020, 2023, 2024, or 2020_moved_2024 |
| `latitude`, `longitude` | WGS84 coordinates (centroid from MIRTA polygon) |
| `conus` | Y/N — continental United States |
| `mirta_match` | Matched MIRTA feature name |

Geocoding source: MIRTA DoD Sites Boundaries GeoJSON (DataLumos Project 239602). 187 auto-matched via fuzzy string matching (threshold ≥ 0.60); 43 manually geocoded from known installation coordinates.

### State Restriction Database (`state_restrictions.csv`)

25 states with foreign land ownership restrictions, compiled from the National Agricultural Law Center, CRS, and individual state legislatures.

| Field | Description |
|-------|-------------|
| `state` | State name |
| `first_enacted_year` | Year of first restriction (or "pre-existing") |
| `first_bill` | Bill number |
| `provision_type` | "ban" or "restriction" |
| `target_scope` | "adversarial_nations" or "all_foreign" |
| `target_countries` | Specific countries targeted |
| `divestment_requirement` | Y/N |
| `enforcement_mechanism` | Type of enforcement |

### Counterfactual Jurisdiction Analysis (`cfius_jurisdiction_analysis.csv`)

119 unique counties with Chinese-linked AFIDA holdings, each classified under four regulatory regimes.

| Field | Description |
|-------|-------------|
| `fips` | 5-digit county FIPS code |
| `county`, `state` | County and state names |
| `total_acres` | Total Chinese-linked acreage in county |
| `r{1-4}_covered` | Y/N — within CFIUS jurisdiction under regime |
| `r{1-4}_nearest_site` | Nearest Appendix A site under regime |
| `r{1-4}_nearest_dist` | Distance in miles (Haversine) |
| `r{1-4}_covering_site` | Site providing coverage (if covered) |
| `r{1-4}_covering_threshold` | 1 or 100 miles |

## The Compound Gap

The compound detection probability multiplies two independently measured gaps:

```
P(detection) = P(within CFIUS jurisdiction) × P(identifiable as foreign)
             = 49.6% × 7.3%
             = 3.6%
```

- **Jurisdictional gap** (this paper): 50.4% of counties outside CFIUS perimeter
- **Visibility gap** (Article 2): 92.7% of entities invisible to federal ownership analysis
- **Voluntariness gap**: CFIUS Part 802 filing is voluntary (not quantified)

The 3.6% figure assumes 100% voluntary filing compliance. Actual detection is lower.

## The Part 1/Part 2 Reclassification Finding

The paper's most actionable finding: 30 counties are within 100 miles of a Part 1 installation but outside CFIUS jurisdiction because Part 1 carries only a 1-mile threshold. Reclassifying Part 1 to Part 2 would:

- Increase county coverage: 49.6% → 74.8% (+25.2 pp)
- Increase acreage coverage: 66.9% → 80.9% (+14.0 pp)
- Require: no new installations, no legislation, no appropriations

This single administrative reform would close more of the jurisdictional gap than all three prior Appendix A expansions combined.

## Known Limitations

1. **County centroids** introduce ±10–35 miles of spatial imprecision (Path C)
2. **Cross-sectional design**: 2024 holdings tested against historical regimes
3. **Theoretical jurisdiction**: measures coverage, not actual CFIUS review
4. **AFIDA data quality**: GAO-documented reporting gaps and reclassification issues
5. **Sample size**: N=119 limits statistical power for density and DiD tests
6. **Part 3/4 excluded**: ICBM missile fields and offshore areas not analyzed
